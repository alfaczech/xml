Úvod

Semestrální práce z předmětu Značkovací jazyky se zabývá tvorbou XML dokumentu
popisujícího soubor do kterého smyšlená aplikace ukládá svá nastavení.
Součástí je také tvorba XSD schéma, XSL transformace do HTML a tato dokumentace.

Dokument byl vytvořen jako konfigurační soubor vymyšlené hry

Metodika

Všechny dokumenty byly vytvořeny v programu Visual Studio 2015.

Nejprve byl vytvořen XML dokument se jmény elementů a atributů.
Ten byl následně konvertován do XSD pomocí webové aplikace Freeformater v defaultním nastavení.
Poté byla ručně upravena celková strktura nového dokumentu XSD
a dokument byl doplněn o restrikce na datové typy.

Na základě v předchozím kroku vytvořeného XSD bylo vygenerováno finální XML
pomocí webové aplikace XSD2XML. Některé hodnoty byly manuálně změněny.

Validita těchto dvou dokumentů byla zkontrolována pomocí webového validátoru .
Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu.

XSL transformace do HTML byla psána ručně. Data jsou strukturována do HTML tabulek
podle kategorií vyplývajících z designu XSD dokumentu.

Pro XSL transformaci do JSON byl nejprve z XML vygenerován JSON soubor, který byl použit
jako základ pro transformaci. Všechny elementy, které obsahovaly pouze jeden atribut
a žádný vnořený element, byly přejmenovány podle vzoru element-atribut. Kategorie nastavení
vyplícající ze struktury dokumentu byly zabaleny do for-each cyklů. Validita JSON
souboru vytvořeného transformací byla ověřena na webovém validátoru Json Parser Online

Celý projekt byl nakonec nahrán na GitHubu.

Popis transformace

HTML

Data jsou transformována do HTML dokumentu. Jsou strukturována tak, jak by typicky bylo strukturováno
Nastavení ve hrách. Jednotlivé elementy a atributy jsou zapsány do tabulek. Všechny atributy "enabled"
jsou zabaleny v podmínkách, přes které se přepisují z true/false na On/Off. Transformace od HTML
neobsahuje všechna data z XML souboru.

JSON

Data jsou transformována do JSON dokumentu. Jsou strukturována stejně jako výchozí XML soubor.
Elementy s datovým typem string jsou zapsány v úvozovkách. Číselné datové typy
a boolean jsou zapsnány bez úvozovek.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy.
Jeho výhodou je dobrá čitelnost a široké možnosti filtrování a omezování dat a datovýh typů.
Práce s XML není nákladná, protože k jeho editaci stačí pouze textový editor.
